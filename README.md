# 🎮 Interactive Turtle Shapes

## 📖 Overview
This project was created for fun using **Python’s Turtle graphics library**.  
It allows users to **control movement with arrow keys** and **draw shapes with number keys**.  
The program demonstrates **event-driven programming** and creative coding with Turtle graphics.

---

## ✨ Features
- ⬆️⬇️⬅️➡️ **Movement Controls**: Use arrow keys to move the turtle up, down, left, or right.
- 🟥 **Draw Squares**: Press `4` to draw a red square.
- 🔵 **Draw Circles**: Press `0` to draw a blue circle.
- 🟩 **Draw Triangles**: Press `3` to draw a green triangle.
- 🎨 **Interactive Canvas**: Combine movement and shapes to create unique designs.
